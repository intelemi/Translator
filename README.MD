# Intelemi Translator

## Install dependences 

### Create venv

```python
python -m venv intelemi_translator_venv
```

### Activate venv

win

```cmd
.\intelemi_translator_env\Scripts\activate
```
Linux
```bash
source .\intelemi_translator_env\Scripts\activate
```

### Install dependences

```bash
pip install -r requirements.txt
```
